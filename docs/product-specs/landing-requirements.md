# **个人网站 Landing Page 需求文档（PRD）**  
**最终版 v2.1**

**项目名称**：个人品牌官网首页（Landing Page）  
**版本**：v2.1（中文优化最终版）  
**日期**：2026年4月  
**核心理念**：构建选择权（Building Sovereignty）  
**关键词**：Investing · Self-Training · Sovereignty  
**强调色**：Deep Teal（深青绿） #0A7C7E

## 1. 项目背景与目标
- 参考 Dan Koe 的极简哲学风格，打造一个属于你自己的个人品牌首页。
- 核心信息：帮助人们通过 **Investing（投资自己）**、**Self-Training（系统自我训练）**，最终获得 **Sovereignty（人生选择权）**。
- **页面核心目标**：
  1. 在 10 秒内让访客清晰理解你的核心理念和价值。
  2. 强引导订阅 Substack Newsletter，积累高质量读者资产。
  3. 展示思考深度，建立信任。
  4. 为后续付费产品（课程、系统、咨询等）做好铺垫。
  5. 整体气质：深沉、冷静、有力量，同时保持人文温度，像一本高端的中文哲学思考书。

**重要说明**：网站主要内容为中文，需优先保证中文显示美观、易读，支持良好中英混排。

## 2. 整体设计风格与 UI 要求（Deep Teal 版）
- **模式**：深色模式（Dark Mode）
- **配色方案**：
  - 主背景色：#0F0F0F（深沉黑灰）
  - 主文字色：#F0EDE8（柔和暖白）
  - **强调色（Accent）**：Deep Teal #0A7C7E（深青绿）——用于按钮、链接、标题下划线、小图标
  - Hover 效果：轻微提亮至 #2A9D9F
  - 次要文字色：#A8A8A8（暖灰）
- **字体推荐（中文优化版）**：
  - **标题字体（Headline / 大标题）**：
    - 中文：**Noto Serif SC / Noto Serif TC**（思源宋体）——优雅衬线体，书卷气强，适合哲学主题
    - 英文：**Playfair Display** 或 **Georgia**（优雅 serif）
  - **正文字体（Body / 文章内容）**：
    - 中文：**Noto Sans SC / Noto Sans TC**（思源黑体）——清晰现代，屏幕阅读体验优秀
    - 英文 / 数字：**Inter**（保持现代感和可读性）
  - **实现方式**（推荐 CSS 写法）：
    ```css
    body {
      font-family: "Noto Sans SC", "Noto Sans TC", Inter, system-ui, sans-serif;
    }
    
    h1, h2, .headline {
      font-family: "Noto Serif SC", "Noto Serif TC", "Playfair Display", Georgia, serif;
    }
    ```
  - **说明**：优先从 Google Fonts 或字体托管服务加载思源系列，确保中文优先显示，中英混排协调。

- **视觉风格关键词**：
  - 极简、大留白、呼吸感强
  - 轻微圆角 + 柔和阴影
  - 冷静有力、人文高级，像深夜与智者对谈的数字空间
  - 保证中文长文阅读舒适

## 3. 页面布局结构（从上到下）
1. **Header**（固定顶部导航栏）
2. **Hero**（英雄区 / 首屏）
3. **Blogs / Letters**（深度文章展示区）
4. **About Me**（关于我）
5. **Footer**（页脚，推荐包含）

## 4. 各模块详细需求与文案（中文为主）

**4.1 Header（导航栏）**
- 固定顶部，背景 #0F0F0F + 轻微透明
- 左侧：Logo（你的名字 或 “[你的名字] | 构建选择权”）
- 右侧菜单（中文）：
  - 首页
  - Letters（或 思考长文）
  - 资源
  - 关于我
- 右侧按钮：**「订阅」** 或 **「订阅 Letters」**（Deep Teal 背景）

**4.2 Hero 区（最重要模块）**
- 高度：90–100vh
- 背景：深色渐变（#0F0F0F → #0F1A1A 轻微 teal 调）
- 文案：

**Headline**（大标题）：
构建你的选择权  
**Investing. Self-Training. Sovereignty.**

**Subheadline**（副标题）：
在快速变化的时代，我帮助你通过持续投资自己、系统化的自我训练，最终获得真正属于自己的人生主权——  
时间自由、财务自由、心智自由，不再被外部环境或他人所掌控。

每周我会写一封深度长信，与你分享经过实践检验的框架与思考。

**CTA Buttons**：
- 主按钮（Deep Teal 背景）：**「每周免费订阅 2 封 Letters」**  → 跳转 Substack 订阅
- 次按钮（透明边框）：**「阅读最新思考」** → 平滑滚动到 Letters 区

下方小字：
“选择权不是天生的，而是通过持续的自我投资与训练构建而成。”

**4.3 Blogs / Letters 区**
- 标题（serif）：**Letters · 我的深度思考**
- 副标题：关于 Investing、Self-Training、Sovereignty 的长文。每周更新，帮助你建立清晰有力的认知框架。
- 展示 3–6 篇文章卡片（标题 + 1–2 行摘要 + 日期 + 阅读按钮）
- 下方强 CTA：
  **想系统接收这些思考吗？**
  **「立即订阅我的 Newsletter」**（Deep Teal 按钮 + Substack 表单）

**4.4 About Me 区**
- 标题：**关于我**
- 正文（第一人称）：

嘿，我是 [你的名字]。

我曾经走在传统的路径上，却逐渐发现外部给予的选择越来越少。直到我开始把注意力转向“构建选择权”——通过持续的 Investing（投资时间、精力和资源到自己身上），以及严格而系统的 Self-Training，我才慢慢拿回了对人生的主导权。

现在，我把这些年在实践中踩过的坑、验证过的框架，以及在 AI 时代依然有效的思考，写成一封封 Letters，分享给那些同样渴望拥有更多选择权的人。

这里没有速成的方法，也没有空洞的鸡汤。  
只有经过反复训练才能真正内化的认知，和一步步构建主权的清晰路径。

如果你也在寻找不被时代浪潮左右的人生，欢迎加入这场持续的自我训练。

（建议在此放置一张自然、光线柔和的个人照片）

下方 CTA：
**「准备好开始构建你的选择权了吗？」**  
**订阅 Letters，一起前行。**

## 5. 通用技术与功能要求
- **响应式设计**：手机、平板、桌面端完美适配（中文排版需特别注意行高和字间距）
- **Substack 集成**：Hero 和 Letters 区各放置一次订阅入口
- **性能优化**：首屏优先加载，图片压缩
- **SEO 优化**：
  - Title：构建你的选择权 | Investing. Self-Training. Sovereignty.
  - Description：通过自我投资与系统训练，获得人生主权。每周深度 Letters，助你构建真正属于自己的自由。
- **字体加载**：优先加载 Noto Sans SC / Noto Serif SC

## 6. 实施建议
- **开发优先级**：Hero > Letters > About Me > Header
- **内容准备**：准备 3–5 篇真实中文文章用于展示
